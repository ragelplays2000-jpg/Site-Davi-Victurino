# Esportes Victurino — Site Fotografia Esportiva

Site profissional para Davi Viturino Pinheiro, fotógrafo esportivo na Bahia.

## Estrutura de pastas

```
esportes-victurino/
├── index.html
├── assets/
│   └── img/
│       ├── davi-perfil.jpg
│       ├── casamentos/        casamentos-1.jpg … casamentos-3.jpg
│       ├── aniversarios/      aniversarios-1.jpg … aniversarios-6.jpg
│       ├── gestante/          gestante-1.jpg … gestante-10.jpg
│       ├── corporativo/       corporativo-1.jpg … corporativo-3.jpg
│       └── esportes/
│           ├── corrida/       corrida-1.jpg … corrida-7.jpg
│           └── futebol/       futebol-1.jpg … futebol-20.jpg
```

## Como adicionar as fotos

Nomeie os arquivos exatamente como indicado acima (ex: `futebol-1.jpg`, `futebol-2.jpg`...).  
A galeria carrega os caminhos automaticamente pelo JavaScript — nenhuma alteração no HTML é necessária.

## Deploy

1. Adicione as fotos nas pastas corretas.
2. Arraste a pasta `esportes-victurino/` para o [Netlify Drop](https://app.netlify.com/drop) ou [Cloudflare Pages](https://pages.cloudflare.com/).
3. (Opcional) Configure um domínio customizado no painel do serviço escolhido.
